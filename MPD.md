## ncmpc

ncmpc is a client for MPD.

### Basic controls

* `Shift-P` - pause
* `s` - stop
* `left arrow` - vol down
* `right arrow` - vol up
* `c` - clear playlist
* `a` - add to playlist
* `t` - add all to playlist
* `y` - single mode (don't play any songs after current one stops)
* `z` - random mode
* `C` - consume mode (capital C). Deletes song from playlist after playing.
