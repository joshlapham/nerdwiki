### Add user

Use with `-m` option to create home folder -

`useradd nameOfUser -m`

### Remove a user

Use with `-r` option to remove home folder and mail spool -

`userdel -r nameOfUser`

### Add user to group

`adduser nameOfUser nameOfGroup`

### Add user to `sudoers`

`adduser nameOfUser sudo`

### `uid`/`gid` info

`id nameOfUser`
